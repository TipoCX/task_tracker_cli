# Task Tracker CLI

proyect idea taken from roadmap practice proyects:
https://roadmap.sh/projects/task-tracker

## instructions
### add task
``` add <description> ```
all tasks will be defaulted to todo status

### list tasks
```list```
#### filtered tasks list by status
```list <status>```
- todo
- in-progress
- done
  
### update task
```update <task id> <description>```

### change task status
``` mark-todo <task id> ```

``` mark-in-progress <task id> ```

``` mark-done <task id> ```

### delete task
``` delete <task id> ```
