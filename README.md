# Task Tracker CLI

## instructions
### add task
``` add <description> ```
all tasks will be defaulted to todo status

### list tasks
```list```
#### filtered tasks list by status
```list <status>```
- todo
- in-progress
- done
  
### update task
```update <task id> <description>```

### change task status
``` make-todo <task id> ```

``` make-in-progress <task id> ```

``` make-done <task id> ```

### delete task
``` delete <task id> ```
